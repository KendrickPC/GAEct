Fun With Coding

Beginners

Fun with coding is an in class experience where students use English to gain emotional awareness as well as to learn basic coding skills using MIT's software Scratch Jr.

This course is an activity based learning program. Children will be helping characters from our comic book progress through levels by participating in classroom activities and creating programs.
